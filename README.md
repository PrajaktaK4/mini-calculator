# Dusky Mini Calculator 🌙

A clean, moody-themed calculator built with HTML, CSS, and JavaScript.

### ✨ Features
- Arithmetic operations
- Keyboard support
- History log of previous calculations
- Light/Dark theme toggle
- Fully responsive layout
